# vue-ts-vuetify

## Features

- Develop Framework: Vue 2 + Vue-Router + Vuex
- UI Framework: Vuetify
- Type Check: Typescript
- Code Style: ESlint
- Pre-commit Check: Husky

## Commands

Project setup

```
npm install
```

Compiles and hot-reloads for development

```
npm run serve
```

Compiles and minifies for production

```
npm run build
```

Lints and fixes files

```
npm run lint
```
